# 小红书发布文案

## 标题

本地 html 网页，怎么变成可分享链接？

## 正文

本地做了个网页，却发不出去？

关卡 1：本地 html 别人根本打不开，发文件夹、压缩包都麻烦。  
关卡 2：网上也有方案，像 GitHub Pages、Vercel、腾讯云 COS，都能挂静态网页；只是步骤多一点，刚接触不太好上手，有时还会访问慢、打不开。

一开始，是想给刚接触 vibe coding 的朋友用的，就自己搭了个平台——HTMLHub。  
国内服务器，上传文件或文件夹，生成链接就能发给朋友看；也能粘贴代码，还能更新页面、切换公开或私密。

适合练手网页、AI 生成的小页面、静态 html 项目，想快速预览作品的时候。  
也是慢慢积累了一百用户了，欢迎前来体验哦～

具体怎么用看图就行。入口我放评论区啦。
