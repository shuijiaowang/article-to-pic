# 怎么把本地html转为链接可分享

## 封面区

自己写了个网页、做了个小项目，想发给朋友看看，  
结果发现本地 `html` 文件根本没法直接分享？

这时候，你可以试试我的网站：  
`https://htmlhub.lyyxy.top/home/upload`

它可以上传 `html` 文件，或者直接上传整个项目文件夹，  
然后生成一个可以打开的链接，别人点开就能看。

## 正文区

很多刚接触 vibe coding 的朋友，都会遇到一个很实际的问题：  
页面已经做出来了，但不知道怎么把它变成一个能发给别人看的网址。

有些人会去用国外的平台，思路其实没问题，  
也是上传代码后返回一个可访问的域名链接。  
但这类平台经常会遇到两个问题：

1. 访问速度慢
2. 有时候甚至打不开

如果你只是想把自己做的小页面、小 demo、练手作品快速分享给朋友，  
这样的体验其实不太友好。

所以我做了 `HTMLHub`。

它的用法很简单：

1. 打开上传页面
2. 上传一个 `html` 文件，或者整个项目文件夹
3. 等系统处理完成
4. 拿到一个可直接访问的链接
5. 发给朋友查看

![image](./assets/image.png "width=539 height=843")

相比一些国外服务，`HTMLHub` 的服务器在国内，  
正常访问会更稳一些，打开速度也更适合日常分享。

而且现在还有免费额度，  
对于刚入门、想先把作品发出去看看效果的人来说，已经够用了。

如果你现在正好做了一个静态网页，  
又不想折腾部署流程，  
那它会是一个很省事的选择。
