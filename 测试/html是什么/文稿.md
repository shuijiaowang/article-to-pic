### 第 2 页 · HTML 长什么样子 == Word

像是你写Word一样，`标题，正文，图片`

```
我的主页 
欢迎来到我的主页，我是睡觉王
一张照片.jpg
```
但是这样吧，但是这样吧，计算机也不知道哪个是标题，哪个是正文，傻傻分不清。
一个很简单的方法就是给它标起来，框起来，左右包围。

```
<标题> 我的主页  </标题>
<段落> 欢迎来到我的主页，我是睡觉王 </段落>
<图片> 一张照片 </图片>
```

改成英文：↓

```
<h1>我的主页</h1>
<p>欢迎来到我的主页，我是睡觉王</p>
<img src="头像.jpg">
```

渲染出来：标题要字大加粗，正文要普普通通，还有图片要居中对齐

### 第 3 页 · CSS == 化妆

人靠衣装马靠鞍，html离不开css

默认的呢不能说很丑陋，就是简洁，你想让它更好看，就得多费口舌。

口舌呢就是：字号、颜色、字体、对齐、边距、圆角……（想改啥就写啥）

<标题：居中，字号45，楷体，黑色> 我的主页 </标题>
<段落：宋体，字号18，灰色，行距松一点> 欢迎来到我的主页，我是睡觉王 </段落>
<图片：宽500，高500，圆角，居中> 一张照片 </图片>

改成英文：↓

```
<h1 align=center size=45 font=楷体 color=black>我的主页</h1>
<p font=宋体 size=18 color=gray line-height=松>欢迎来到我的主页，我是睡觉王</p>
<img src="头像.jpg" width=100 height=100 round=12 align=center>
```

这样呢就会好看一点

### 第 4 页 · JS == 动起来

html是骨架，css是画皮，js就是走起来

最常见的js的那肯定就是按钮了：

按钮-点赞/收藏/加关注

<标题> 我的主页  </标题>
<段落> 欢迎来到我的主页，我是睡觉王 </段落>
<按钮> 点赞 </按钮>
<按钮> 收藏 </按钮>
<按钮> 加关注 </按钮>

js说:
    点击了点赞按钮，点赞变成红色
    收藏，收藏变成黄色
    加关注，把字改成已关注

### 第 5 页 · 开始vibecoding做html
前几页呢，只是当做简单的科普用，也是十分的不规范，

当你在用如codex等做网页时，可以像盖房子一样一步一步来：
html：先描述要建几个房间，哪里是门那里是窗，大致长宽高，每个房间主要是做什么的，水电布置
js:有什么功能，睡觉的，做饭的，洗澡的，，，等
css:装修呢最后做就可以，一点都不急，

例如：
例如：还是「睡觉王」那个主页，跟 AI 可以这么分段说——

**第 1 步 · 只要 HTML（先盖框架，丑没关系）**
> 做一个个人主页：顶部大标题「睡觉王的主页」，
> 下面一段自我介绍，中间放头像，
> 底部三个按钮：点赞、收藏、加关注。
> 先不要管好看，黑白简洁就行。

**第 2 步 · 再加 JS（装功能）**
> 点赞点了变红，收藏点了变黄，
> 加关注点了改成「已关注」。

**第 3 步 · 最后 CSS（装修，一点都不急）**
> 标题居中、背景淡色、按钮圆角，
> 整体清爽一点，像教程长图那种风格。


### 第 6 7 页 · 跟 AI 说话前先认认 UI 词


| 词 | 人话（你看到的） | 跟 AI 可以这么说 |
|----|------------------|------------------|
| **导航栏 / 路由** | 页面最上面一排，点「首页」「关于我」切换不同页 | 「顶部加一个导航栏，三个 Tab：主页、作品、联系我，点了换页面」 |
| **菜单** | 一串可点的选项，横着或竖着排 | 「左侧做一个菜单：首页、设置、退出」 |
| **下拉菜单** | 点一下弹出一列选项，不点就收起来 | 「右上角用户名，点开下拉菜单：个人资料、退出登录」 |
| **轮播图** | 几张图自动/手动切换，像幻灯片 | 「首页顶部放轮播图，3 张 banner，能左右切换」 |
| **表格** | 横竖格子排数据，像 Excel | 「用表格展示：姓名、粉丝数、更新时间，三列五行」 |
| **卡片** | 一块一块的内容盒子，带边框或阴影 | 「作品区用卡片布局，每张卡片：封面图 + 标题 + 简介」 |
| **弹窗** | 点按钮后中间蹦出来的小窗口 | 「点「加关注」弹出弹窗，里面写一句感谢」 |
| **侧边栏** | 页面左右固定一条，放菜单或辅助信息 | 「左侧侧边栏放导航：主页、睡过清单、设置」 |
| **搜索框** | 输入关键字查内容 | 「顶部加搜索框，输入地名筛选睡过清单」 |
| **分页器** | 数据太多，底部分页按钮 1、2、3、下一页 | 「表格下方加分页器，每页 10 条」 |
| **折叠面板** | 点标题展开 / 收起内容 | 「FAQ 用折叠面板：点问题展开答案」 |
| **抽屉** | 从屏幕边缘滑出一块面板 | 「点筛选按钮，右侧抽屉滑出筛选条件」 |
| **步骤条** | 分几步完成，显示当前第几步 | 「注册分三步，顶部步骤条：填资料 → 验证 → 完成」 |
| **工具栏** | 一排操作按钮或图标 | 「列表上方工具栏：新增、删除、导出」 |
| **徽标 / 红点** | 图标右上角小数字或未读点 | 「消息图标加红点，显示未读 3 条」 |
| **悬浮按钮** | 右下角圆形固定按钮 | 「右下角悬浮按钮 +，点一下新增睡过记录」 |

### 第 7 页 · 常见的 HTML 标签

| 你看到的 | 人话 |
|----------|------|
| `<h1>` `<h2>` | 标题，数字越小字越大（`h1` 最大） |
| `<p>` | 普通段落 |
| `<img>` | 图片，`src=` 后面写图片文件名 |
| `<a>` | 链接，点了跳转到别的页或别的网站 |
| `<button>` | 按钮，点了要发生点啥（配合 JS） |
| `<div>` | 万能容器，把一块内容包在一起（布局常用） |
| `<ul>` `<li>` | 无序列表，一行一条，前面带小圆点 |
| `<input>` | 输入框，填昵称、密码、搜索词那种 |
| `<table>` `<tr>` `<td>` | 表格：整张表 / 一行 / 一格 |
| `<span>` | 行内小容器，包几个字单独改样式 |
| `<form>` | 表单，把输入框、按钮包在一起提交 |
| `<textarea>` | 多行输入框，写长留言、长描述 |
| `<select>` `<option>` | 下拉选择框，点选其中一个选项 |
| `<ol>` `<li>` | 有序列表，一行一条，前面带 1、2、3 |

### 第 8 页 · 常见的 CSS 属性

| 你看到的 | 人话 | 作用 |
|----------|------|--------|
| `color` | 字体颜色 | 字是黑是红是灰 |
| `font-size` | 字号 | 字大字小 |
| `font-family` | 字体 | 楷体、宋体、系统默认 |
| `font-weight` | 字重 | 普通 / 加粗（bold） |
| `text-align` | 对齐 | 左对齐、居中、右对齐 |
| `background-color` | 背景色 | 整块区域的底色 |
| `width` / `height` | 宽 / 高 | 盒子或图片多大 |
| `margin` | 外边距 | 盒子与盒子之间的距离 |
| `padding` | 内边距 | 文字与边框之间的距离 |
| `border-radius` | 圆角 | 按钮、头像、卡片四角变圆 |
| `display: flex` | 弹性布局 | 几个东西横着排、竖着排、居中 |
| `line-height` | 行高 | 行与行之间松还是紧 |
| `border` | 边框 | 盒子外围一条线 |
| `opacity` | 透明度 | 0 全透明，1 不透明 |
| `box-shadow` | 阴影 | 卡片、按钮浮起来的感觉 |
| `overflow` | 溢出处理 | 超出隐藏或出现滚动条 |

### 第 9 页 · 常见的 JS 说法

鼠标事件：左击右击，双击，按下-按着-松开
鼠标悬浮：鼠标移上去时触发（变色、出提示）
鼠标拖拽：按住鼠标拖着移动（拖滑块、拖上传区）
键盘事件：按下某个键时触发（回车提交、Esc 关闭）
定时任务：到点了自动执行一次（3 秒后弹提示）
循环任务：隔一段时间重复执行（轮播图每 5 秒切一张）
导入数据：从文件或接口把数据读进页面
导出数据：把页面数据下载成 CSV / Excel 等文件
懒加载：图片没出现在屏幕里时不加载
数据持久化：刷新/下次打开还在
页面跳转：从当前页跳到另一个地址
上传文件：选择文件上传/拖拽上传
分页加载：数据太多拆成多页，点下一页再看
无限滚动：滚到页面底部自动加载下一批
动画过渡：元素平滑移动、淡入淡出
弹窗提示：操作完弹出轻提示（成功 / 失败）
禁止点击：不满足条件不让点
