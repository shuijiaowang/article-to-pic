第 1 课

HTML
到底是什么？

做网页为什么离不开它？
零基础也能听懂的傻子式科普 👇

HTML 长什么样子 == Word

像是你写 Word 一样：标题、正文、图片，随手一写——

我的主页 欢迎来到我的主页，我是睡觉王 一张照片.jpg

但计算机也分不清哪个是标题、哪个是正文，傻傻分不清 😵

很简单：给它标起来，框起来，左右包围——

<标题> 我的主页 </标题> <段落> 欢迎来到我的主页，我是睡觉王 </段落> <图片> 一张照片 </图片>

改成英文 ↓ 这就是 HTML 标签

<h1>我的主页</h1> <p>欢迎来到我的主页，我是睡觉王</p> <img src="头像.jpg">

▼ 浏览器渲染出来 ▼

我的主页

欢迎来到我的主页，我是睡觉王

🖼️

标题 → 字大加粗 · 正文 → 普普通通 · 图片 → 居中显示

第 2 页

CSS == 化妆 💄

人靠衣装马靠鞍，HTML 离不开 CSS。默认样式不能说丑，就是太简洁。想好看？得多费口舌——

字号、颜色、字体、对齐、边距、圆角……想改啥就写啥

<标题：居中，字号45，楷体，黑色> 我的主页 </标题> <段落：宋体，字号18，灰色，行距松一点> 欢迎来到… </段落> <图片：宽500，高500，圆角，居中> 一张照片 </图片>

改成英文 ↓

<h1 align=center size=45 font=楷体>我的主页</h1> <p font=宋体 size=18 color=gray>欢迎来到…</p> <img width=100 height=100 round=12>

▼ 加上 CSS 后 ▼

我的主页

欢迎来到我的主页，我是睡觉王

😴

这样呢，就会好看一点 ✨

第 4 页

开始 vibecoding 做 HTML 🏠

前几页只是科普，十分不规范。用 Codex 等 AI 做网页，可以像盖房子一步一步来：

HTML → 先描述几个房间、门窗、水电
JS → 有什么功能：睡觉、做饭、洗澡…
CSS → 装修最后做，一点都不急

还是「睡觉王」主页，跟 AI 可以这么分段说 👇

第 1 步 · 只要 HTML（先盖框架，丑没关系）
做一个个人主页：顶部大标题「睡觉王的主页」，下面自我介绍，中间放头像，底部三个按钮：点赞、收藏、加关注。先不要管好看，黑白简洁就行。
第 2 步 · 再加 JS（装功能）
点赞点了变红，收藏点了变黄，加关注点了改成「已关注」。
第 3 步 · 最后 CSS（装修，一点都不急）
标题居中、背景淡色、按钮圆角，整体清爽一点，像教程长图那种风格。

💡 记住顺序：HTML → JS → CSS，别一上来就纠结好不好看！

第 5 页

跟 AI 说话前先认认 UI 词

词人话（你看到的）跟 AI 可以这么说
导航栏页面最上面一排，点 Tab 切换不同页「顶部加导航栏，三个 Tab：主页、作品、联系我」
菜单一串可点的选项，横着或竖着排「左侧做菜单：首页、设置、退出」
下拉菜单点一下弹出一列选项「右上角用户名，点开下拉：个人资料、退出登录」
轮播图几张图自动/手动切换，像幻灯片「首页顶部放轮播图，3 张 banner，能左右切换」
表格横竖格子排数据，像 Excel「用表格展示：姓名、粉丝数、更新时间，三列五行」
卡片一块一块的内容盒子，带边框或阴影「作品区用卡片：封面图 + 标题 + 简介」
弹窗点按钮后中间蹦出来的小窗口「点加关注弹出弹窗，写一句感谢」
侧边栏页面左右固定一条，放菜单或辅助信息「左侧侧边栏：主页、睡过清单、设置」
搜索框输入关键字查内容「顶部加搜索框，输入地名筛选睡过清单」
分页器底部分页按钮 1、2、3、下一页「表格下方加分页器，每页 10 条」
折叠面板点标题展开 / 收起内容「FAQ 用折叠面板：点问题展开答案」
抽屉从屏幕边缘滑出一块面板「点筛选按钮，右侧抽屉滑出筛选条件」
步骤条分几步完成，显示当前第几步「注册分三步，顶部步骤条：填资料 → 验证 → 完成」
工具栏一排操作按钮或图标「列表上方工具栏：新增、删除、导出」
徽标/红点图标右上角小数字或未读点「消息图标加红点，显示未读 3 条」
悬浮按钮右下角圆形固定按钮「右下角悬浮按钮 +，点一下新增睡过记录」
第 6 页

常见的 HTML 标签

你看到的人话
<h1> <h2>标题，数字越小字越大（h1 最大）
<p>普通段落
<img>图片，src= 后面写图片文件名
<a>链接，点了跳转到别的页或网站
<button>按钮，点了要发生点啥（配合 JS）
<div>万能容器，把一块内容包在一起
<ul> <li>无序列表，一行一条，前面带小圆点
<input>输入框，填昵称、密码、搜索词
<table> <tr> <td>表格：整张表 / 一行 / 一格
<span>行内小容器，包几个字单独改样式
<form>表单，把输入框、按钮包在一起提交
<textarea>多行输入框，写长留言、长描述
<select> <option>下拉选择框，点选其中一个选项
<ol> <li>有序列表，前面带 1、2、3

常见的 CSS 属性

你看到的人话作用
color字体颜色字是黑是红是灰
font-size字号字大字小
font-family字体楷体、宋体、系统默认

opacity透明度0 全透明，1 不透明

第 8 页

常见的 JS 说法

  • 鼠标事件：左击、右击、双击、按下-按着-松开
  • 鼠标悬浮：鼠标移上去时触发（变色、出提示）
  • 鼠标拖拽：按住拖着移动（拖滑块、拖上传区）
  • 键盘事件：按下某个键时触发（回车提交、Esc 关闭）
  • 定时任务：到点了自动执行一次（3 秒后弹提示）
  • 循环任务：隔一段时间重复执行（轮播图每 5 秒切一张）
  • 导入 / 导出数据：从文件读进页面 / 下载成 CSV、Excel
  • 懒加载：图片没出现在屏幕里时不加载
  • 数据持久化：刷新 / 下次打开还在
  • 页面跳转：从当前页跳到另一个地址
  • 上传文件：选择文件上传 / 拖拽上传
  • 分页加载：数据太多拆成多页，点下一页再看
  • 无限滚动：滚到底部自动加载下一批
  • 动画过渡：元素平滑移动、淡入淡出
  • 弹窗提示：操作完弹出轻提示（成功 / 失败）
  • 禁止点击：不满足条件不让点
