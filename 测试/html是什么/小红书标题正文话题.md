## 标题

HTML 是什么？网页开发的起点

## 正文

做网页，为什么离不开 HTML？

HTML 是网页的结构层，负责定义标题、段落、图片等内容；CSS 负责样式，JS 负责交互。三者分工明确，缺一不可。

用 AI 做网页时，建议分三步：先搭 HTML 框架，再加 JS 功能，最后做 CSS 美化。
