📖 分页交互示例

预览模式下可点击分页按钮切换内容，再切回编辑模式继续修改

👋 欢迎使用可视化编辑

这是一个带 JavaScript 分页的示例页面。点击上方标签或下方「上一页 / 下一页」即可切换内容。

在「预览模式」里页面交互完全可用；切到「编辑模式」后，可以点选标题、段落、按钮修改文案和样式。

试试点击顶部的「功能介绍」标签，看看分页切换效果。

⚡ 工具能做什么

HtmlHub 可视化编辑是纯前端本地工具，不上传服务器，适合快速改页面。

点选改样式颜色、字号、间距、圆角等
改文案结构标题、段落、按钮文字
预览可交互分页、表单、按钮都能点
多文件缓存上传多个 HTML 随时切换

🛠 三步上手

1
先在预览模式体验分页和页面交互，确认效果是否符合预期。
2
切换到编辑模式，点击要修改的元素，在右侧面板调整样式或文案。
3
编辑完成后点击「下载 HTML」，文件会保存到本地并写入浏览器缓存。

💡 小提示

如果你的页面本身就有 JS 逻辑（比如这个分页），编辑完切回预览模式可以继续测试交互。

工具栏支持添加多个 HTML 文件；每个文件独立缓存，切换时会自动保存当前编辑进度。

觉得示例不合适？点工具栏「恢复示例」可还原此模板。
