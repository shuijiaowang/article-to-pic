第 2 页 · 核心能力

这套分页约定

外部 AI 可生成任意复杂 HTML，只要每页使用 class="page" 且宽高为 1080×1440，即可在本站预览与导出。

自由样式

CSS、渐变、卡片、图标均可组合。

可视化编辑

点选元素改文案、颜色、间距；支持撤销与多版本。

分页导出

预览页按页检测溢出，一键导出高清 PNG 长图。

上传即用

在 Cursor / Codex 生成 HTML 后，上传到文稿管理即可开始。

第 3 页 · 推荐流程

三步上手

你可以直接编辑本示例，或替换为 Cursor 生成的 HTML。

1

外部生成 HTML

在 Cursor 中描述排版需求，生成含多页 .page 的完整 HTML 文件。

2

上传并可视化编辑

文稿管理上传 HTML → 进入本页微调文案与样式 → 保存修改。

3

预览页导出 PNG

返回预览检查分页与溢出，确认无误后导出全部 PNG。

提示：编辑模式改 inline 样式；预览模式可滚动查看各页实际效果。保存后返回预览页即可导出图片。
